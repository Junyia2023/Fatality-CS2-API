## control_id

此类型表示一个控件ID。

## __call

[![构造函数][这是此类型的构造函数定义。]rw]

构造ID结构。

**参数**

| 名称 | 类型 | 描述 |
| ---- | ---- | ----------- |
| `id` | `string` | ID。 |

**返回值**

| 名称 | 描述 |
| ---- | ----------- |
| `control_id` | ID结构。 |

**示例**

```lua
local id = gui.control_id('my_id');
```
