## chandle

此类型表示一个实体句柄。

> 您也可以使用 `==` 运算符比较此类型。

## get

[[字段]]

返回实体，如果未找到则返回 nil。

**参数**

无。

**返回值**

| 类型 | 描述 |
| ---- | ----------- |
| `<type>` | 实体实例。 |

**示例**

```lua
local ent = handle:get();
```

## valid

[[字段]]

如果句柄有效，则返回 `true`。

**参数**

无。

**返回值**

| 类型 | 描述 |
| ---- | ----------- |
| `bool` | 如果有效则为 `true`。 |

**示例**

```lua
if handle:valid() then
    -- ...
end
```

## is_clientside

[[字段]]

如果句柄链接到客户端实体，则返回 `true`。

**参数**

无。

**返回值**

| 类型 | 描述 |
| ---- | ----------- |
| `bool` | 如果是客户端的则为 `true`。 |

**示例**

```lua
if handle:is_clientside() then
    -- ...
end
```
