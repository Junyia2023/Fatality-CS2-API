## vec2

此类型是渲染系统中使用的二维向量。

## __call

[![Constructor][这是此类型的构造函数定义]rw]

创建一个新的二维向量实例。

**参数**

*1. 默认向量 (0, 0).* 

无参数。

*2. 单个值.*

| 名称 | 类型 | 描述 |
| ---- | ---- | ----------- |
| `value` | `float` | X 和 Y 坐标。|

*3. XY 值.*

| 名称 | 类型 | 描述 |
| ---- | ---- | ----------- |
| `x` | `float` | X 坐标。|
| `y` | `float` | Y 坐标。|

**返回值**

| 类型 | 描述 |
| ---- | ----------- |
| `vec2` | 新的向量。|

**示例**

```lua
local vec = draw.vec2(5, 10);
```
